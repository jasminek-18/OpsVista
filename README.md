# OpsVista — Enterprise Operations Intelligence Platform

> End-to-end Data Analytics project simulating a manufacturing company's 
> operations platform covering Procurement, Inventory, Supplier Performance 
> and Logistics Analytics.

## Tech Stack
| Tool | Purpose |
|---|---|
| Python | Data generation, cleaning, EDA, predictive analytics |
| MySQL | Relational schema, ETL, 45+ SQL queries |
| Power BI | 4-page interactive dark-theme dashboard |
| pandas / numpy | Data manipulation and feature engineering |
| matplotlib | EDA visualizations |

## Dataset
- 6,000 Purchase Orders
- 12,000 Sales Orders  
- 9,600+ Shipments
- 80 Suppliers · 10 Warehouses · 600 Customers · 68 Products

## Project Modules
| Module | Description |
|---|---|
| Data Generation | 11 CSVs · 47,000+ rows · realistic business patterns |
| Data Cleaning | Null handling · deduplication · feature engineering |
| SQL Analytics | 45 business queries · CTEs · window functions · rankings |
| Python EDA | 10 Matplotlib charts saved as PNG |
| Predictive Analytics | Vendor risk scoring · reorder prediction · AI recommendations |
| Power BI Dashboard | 4-page dark theme · KPIs · conditional formatting |

## Dashboard Pages
1. Executive Dashboard — KPIs, procurement trends, regional analysis
2. Supplier Performance — Scorecard, delay analysis, spend ranking
3. Inventory & Warehouse — Utilization, stock health, reorder alerts
4. Predictive Analytics — Vendor risk scores, reorder predictions, AI recommendations

## Key Business Insights
- 20 suppliers identified as high-risk (delay rate > 35%)
- 15 SKUs at zero stock requiring emergency purchase orders
- Mumbai and Delhi warehouses above 90% capacity
- East region records highest shipment delay rate (25%)
- Return rate at 7% — above industry threshold of 5%
